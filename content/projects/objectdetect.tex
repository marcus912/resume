\textbf{ObjectDetectBot}\\
Developed an AI-powered desktop application using YOLOv8 for real-time object detection to identify on-screen targets in gaming environments, integrated with Python for screen capture and automated mouse/keyboard controls. Designed and trained a custom YOLOv8 model to achieve high-accuracy target detection. Demonstrated expertise in computer vision and automation, delivering a scalable solution for dynamic screen-based tasks.
